# 🤖 RAG Chatbot with LangChain
Is project mein maine ek advanced chatbot banaya hai jo PDFs ya Text files se context lekar sahi jawab deta hai.
### ✨ Key Features:
* **PDF Querying:** Kisi bhi document se sawal-jawab.
* **Vector Database:** Context store karne ke liye (ChromaDB/Pinecone) use kiya hai.
* **Smart Retrieval:** LangChain use karke LLM (OpenAI/Ollama) ko connect kiya.
